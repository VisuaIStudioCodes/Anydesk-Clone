# Anydesk-Clone

Anydesk-Clone is a lightweight proof-of-concept repository intended to explore remote-desktop style UX patterns and project structure. It provides a minimal baseline so contributors can iterate quickly on UI, networking, and streaming experiments.

## Goals
- Establish a clean starting point for experimentation.
- Document the intended direction and contribution expectations.
- Keep setup and onboarding friction low.

## Getting Started
1. Clone the repository.
2. Add your experimental modules or prototypes.
3. Keep changes scoped and documented in pull requests.

## Contributing
- Keep PRs focused and describe changes clearly.
- Prefer small, incremental improvements.
